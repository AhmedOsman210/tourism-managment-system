Tourism Management System (TMS)



📌 Overview

The Tourism Management System (TMS) is a web-based application that simplifies tourism operations, allowing users to book tours, manage their accounts, and for admins to handle bookings efficiently.

🚀 Features

🎯 User Features

📝 User Registration and Login

📌 View & Book Tour Packages

💳 Online Payment Integration (PayPal, Stripe, etc.)

📜 View Booking History

⭐ Leave Reviews & Ratings

📞 Contact Support

🛠 Admin Features

📊 Admin Dashboard

✏️ Manage Tour Packages (Add, Edit, Delete)

👥 Manage Users (View, Delete)

📅 View & Manage Bookings

📈 Generate Reports

⚙️ Configure Website Settings

✨ Additional Features

🔒 Secure Authentication System

📧 Email Notifications (Booking Confirmations, Password Reset)

📱 Fully Responsive Design

🔎 Search & Filter Tour Packages

🌍 Multi-language Support

🛠 Installation & Configuration

🔽 1. Download & Extract

Download and unzip the file on your local system.

Copy the tms folder into your web server's root directory:

XAMPP: xampp/htdocs

WAMP: wamp/www

LAMP: var/www/html

🛢 2. Database Configuration

Open PHPMyAdmin.

Create a new database named tms.

Import the tms.sql file (found in the zip package).

🌍 3. Running the Application

Open your browser and navigate to:

User Interface: http://localhost/tms/

Admin Panel: http://localhost/tms/admin/
